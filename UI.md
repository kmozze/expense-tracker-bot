# План UI
### 1. /start
Если у пользователя ещё нет категорий, создаются базовые категории (Еда, Транспорт, Жильё, Развлечение, Прочее) и появляется сообщение

```✅ Созданы базовые категории для тебя!```

Затем открывается главное меню
### 2. Главное меню (Reply Keyboard)

При `/start` и после каждого успешного действия бот будет отправлять **Reply Keyboard** (persistent menu)
➕ Добавить расход
📋 Расходы
📂 Категории
📊 Статистика

### 3. Управление категориями

**Команда/кнопка:** `📂 Категории`

```
[ ➕ Добавить новую ]

Еда и напитки     [✏️] [🗑️]
Транспорт         [✏️] [🗑️]
...
```


**Логика удаления:**
1. Проверяем есть ли траты в категории
2. Если **нет** трат → сразу удаляем + подтверждаем.
3. Если **есть** траты → бот отправляет новое сообщение:

```
По категории «Еда и напитки» есть 12 трат на сумму 8450 ₽.

Что сделать?
[🗑️ Удалить категорию и все траты]
[🔄 Перенести траты в другую категорию]
[❌ Отмена]
```

Если выбран перенос — показываем inline-кнопки со всеми остальными категориями.

### 4. Добавление расхода (Свободный ввод)
Основной способ — ввод суммы и комментария одним сообщением. Бот автоматически отделяет число от текста. Текст становится **описанием**, а категорию пользователь выбирает кнопкой.

**Логика работы:**
1. **Ввод:** Пользователь пишет `500 такси` или `обед 450`.
2. **Выбор категории (Inline Keyboard):** Бот парсит сумму и описание, затем предлагает выбрать категорию из списка:

``` 💰 **500 ₽**
   📝 такси
   Куда запишем?
   [Транспорт] [Еда]
   [Жилье] [Другое]
   [ ❌ Отмена ]
   ```

3. **Результат:** После клика расход сохраняется, бот выводит подтверждение:
```
    ✅ Сохранено!
    💰Сумма: 500 ₽
    📂Категория: Транспорт
    📝Описание: такси
    [✏️ Изменить] [🗑️ Удалить]
```
4. При нажатии кнопок под сохраненным расходом:

**При [🗑️ Удалить]:** Бот удаляет запись и заменяет сообщение на:
  `Расход удален`

**При [✏️ Изменить]:** Бот предлагает выбрать поле для правки:

```
    Что изменить?
    [💰 Сумма] [📂 Категория] [📝 Описание]
```
5. Логика изменения конкретных полей:
* **[💰 Сумма]:** Бот просит: `Введите новую сумму`.
    * Если введено число: обновляет расход и возвращает «карточку» (п. 3).
    * Если введена ошибка (текст): пишет `❌ Введите только число` и ждет повторный ввод.
* **[📂 Категория]:** Бот показывает список всех категорий кнопками. После клика по новой категории обновляет расход и возвращает «карточку».
* **[📝 Описание]:** Бот просит: `Введите новое описание`. Любой присланный текст становится описанием, после чего бот возвращает «карточку».

**Общие правила:**
* Любое действие можно отменить кнопкой **[🔙 Назад]**, вернувшись к просмотру расхода.
* После каждой успешной правки бот обновляет «карточку» актуальными данными.


### 5. Просмотр расходов + редактирование/удаление

**Кнопка:** `📋 Расходы`

Появляется меню настройки

```
Настройка отображения 📋

📅 Период: За месяц
📂 Категория: Все

[Изменить период 🗓] [Изменить категорию 📁]
[🔎 Показать список]
```

Дальше:

* **[Изменить период 🗓]:** Бот выводит список быстрых фильтров кнопками:
```
    Выберите период:
    [📅 День] [📅 Неделя] [📅 Месяц] [📅 Все время]
```
После выбора бот обновляет текст меню «Настройка отображения».

* **[Изменить категорию 📁]:** Бот выводит список всех категорий пользователя с дополнительным выбором всё:
```
    Выберите категорию для фильтра:
    [Все] [Еда] [Транспорт] ...
```
После выбора бот обновляет текст меню и фильтрует будущий список.

**Просмотр списка [🔎 Показать список]:**
После выбора бот отправляет список (максимум 20 записей, с пагинацией при необходимости):

```
📋 Расходы за неделю (4 шт.)

1. 12.03  Еда    450 ₽   обед с коллегами
   [✏️ Изменить] [🗑️ Удалить]

2. 11.03  Транспорт         120 ₽   такси
   [✏️] [🗑️]

...
[⬅️ Предыдущие 20]   [Следующие 20 ➡️]
```

- **При редактировании** - та же логика, что была в пункте 4

- **При удалении** подтверждение «Точно удалить?»

### 6. Статистика

**Кнопка:** `📊 Статистика`

Появляется стартовое меню

```
📊 Панель статистики

🗓 Период: Текущий месяц
📁 Категории: Все категорию

[Изменить период 🗓] [Изменить категорию 📁]
================================
[📝 Текстовая сводка]
```

Примеры вывода:

**Текстовая сводка:**
```
📊 За март 2026 — 18 450 ₽

Еда и напитки      8 200 ₽ (44%)
Транспорт          3 900 ₽ (21%)
Жильё              2 500 ₽ (14%)
...
```
