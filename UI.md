# План UI
### 1. /start
Если у пользователя ещё нет категорий, создаются базовые категории и появляется сообщение

```✅ Созданы базовые категории для тебя!```

Затем открывается главное меню
### 2. Главное меню (Reply Keyboard)

При `/start` и после каждого успешного действия бот будет отправлять **Reply Keyboard** (persistent menu)
➕ Добавить расход
📋 Расходы
📂 Категории
📊 Статистика

### 3. Управление категориями

**Команда/кнопка:** `📂 Категории`

```
[ ➕ Добавить новую ]

Еда и напитки     [✏️] [🗑️]
Транспорт         [✏️] [🗑️]
...
```


**Логика удаления:**
1. Проверяем есть ли траты в категории
2. Если **нет** трат → сразу удаляем + подтверждаем.
3. Если **есть** траты → бот отправляет новое сообщение:

```
По категории «Еда и напитки» есть 12 трат на сумму 8450 ₽.

Что сделать?
[🗑️ Удалить категорию и все траты]
[🔄 Перенести траты в другую категорию]
[❌ Отмена]
```

Если выбран перенос — показываем inline-кнопки со всеми остальными категориями.

### 4. Добавление расхода + описание

Оставляем **свободный текст** как основной способ (пользователи любят быстро писать).

Но улучшаем:
- После успешного парсинга и сохранения бот спрашивает:

```
Еда и напитки — 320 ₽
[📝 Добавить описание]
[✅ Сохранить]
[✏️ Изменить]
[❌ Отмена]
```

- **Если «Добавить описание»** — бот ждёт следующее сообщение, оно целиком уйдёт в описание.

- **При редактирование** - бот спрашивает «Новую сумму?», «Новую категорию?» или «Новое описание?» (по шагам или одним сообщением).

- **При удалении** - подтверждение «Точно удалить?»

### 5. Просмотр расходов + редактирование/удаление

**Кнопка:** `📋 Расходы`

Появляется меню настройки

```
Настройка отображения 📋

📅 Период: За месяц
📂 Категория: Все

[Изменить период 🗓] [Изменить категорию 📁]
[🔎 Показать список]
```

После выбора бот отправляет список (максимум 20 записей, с пагинацией при необходимости):

```
📋 Расходы за неделю (4 шт.)

1. 12.03  Еда и напитки     450 ₽   обед с коллегами
   [✏️ Изменить] [🗑️ Удалить]

2. 11.03  Транспорт         120 ₽   такси
   [✏️] [🗑️]

...
[⬅️ Предыдущие 20]   [Следующие 20 ➡️]
```

- **При редактирование** - бот спрашивает «Новую сумму?», «Новую категорию?» или «Новое описание?» (по шагам или одним сообщением).

- **При удалении** подтверждение «Точно удалить?»

### 6. Статистика

**Кнопка:** `📊 Статистика`

Появляется стартовое меню

```
📊 Панель статистики

🗓 Период: Текущий месяц
📁 Категория: Все категории

[Изменить период 🗓] [Изменить категорию 📁]
================================
[📝 Текстовая сводка] [📉 Круговая диаграмма]
[🏆 Топ-5 трат]       [📥 Выгрузить CSV]
```

Примеры вывода:

**Текстовая сводка:**
```
📊 За март 2026 — 18 450 ₽

Еда и напитки      8 200 ₽ (44%)
Транспорт          3 900 ₽ (21%)
Жильё              2 500 ₽ (14%)
...
```
